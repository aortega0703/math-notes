For a [[Set]] $S$, its _identity_ or _diagonal_ function ($\operatorname{id}_S: S \to S$) is defined as$$\operatorname{id}_S(s) =s$$Note that $\operatorname{id}_S$ is an [[Function|inclusion]] with equal domain and codomain.
