For sets $A\subseteq S$ the _inclusion map_$$
\begin{align}
	i_A : A&\to S\\
	a&\mapsto a
\end{align}
$$that sends an object to itself is [[Injectivity and surjectivity|Injective]].
##### Proof:
Let $a,b\in A$ such that $i_A(a) = i_A(b)$, then $a=b$ so $i_A$ is injective.
