For $(n>0) in ZZ$, Euler's totient function $\varphi(n)$ counts the number of positive integers up to $n$ relatively prime to it. That is$$
\varphi(n) = \bigg|\big\{k in ZZ:\ 0<k lt.eq n,\ \gcd(k,n)=1\big\}\bigg|
$$
