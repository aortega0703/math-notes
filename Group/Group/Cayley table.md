A _Cayley table_ for a [[Group]] $G$ displays the element $gf$ in row $g$ and column $f$. This table depends on the order for the rows and columns.

| $\cdot$  | $e$      | $\dots$  | $f$      | $\dots$  |
| -------- | -------- | -------- | -------- | -------- |
| $e$      | e        | $\dots$  | $f$      | $\dots$  |
| $\vdots$ | $\vdots$ | $\ddots$ | $\vdots$ | $\ddots$ |
| $g$      | $g$      | $\dots$  | $gf$     | $\dots$  |
| $\vdots$ | $\vdots$ | $\ddots$ | $\vdots$ | $\ddots$ |
